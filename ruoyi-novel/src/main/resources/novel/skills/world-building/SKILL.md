# 世界观构建技能

## 目标
为网络小说构建完整、可扩展的世界观设定，并**写入数据库**。

## 步骤
1. **首轮必须立即**调用 getProjectInfo 与 getSetting(world)，禁止仅口头说「将要查询数据库」
2. 明确时代背景、地理格局、力量体系（如修炼等级、科技水平）
3. 定义社会结构与主要势力
4. 列出核心规则（什么能做什么不能做）
5. **在同一次或后续对话中，必须调用 saveSetting(settingType=world) 保存完整 Markdown**

## 禁止
- 禁止只回复「接下来我将构建世界观」而不调用 saveSetting
- 禁止输出过短摘要代替完整设定（正文至少数百字）

## 质量要求
- 与项目类型、用户创意一致
- 留出后续情节扩展空间
- 避免与已有设定矛盾
